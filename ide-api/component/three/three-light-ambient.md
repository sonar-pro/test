# AmapThreeLightAmbient
ThreeJS的环境光会均匀的照亮场景中的所有物体

## Attributes

Attribute | Type                                | Description
---|-------------------------------------|---|
color  | String                              | 颜色
intensity | Number                              | 光照的强度。缺省值为 1

## Events

Event Name | Parameters | Description
---|------------|---|
init | Light      | 实例初始化结束
